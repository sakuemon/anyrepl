# anyrepl

A zsh function for any language's repl commands.

# Support languages and repl commands
## php
+ psysh
+ php -a

## ruby
+ pry
+ irb

## node
+ node

# Installation
+ clone this repository to your zsh function directory.
+ autoload -U anyrepl

# Usage
execute php repl command
```
anyrepl php
```

need args?
```
anyrepl php vendor/autoload.php
```
